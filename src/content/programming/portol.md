A personal portfolio site that moves away from traditional portfolios, showcasing different sectors of work through distinct visual modes. Each mode has its own visual identity, layout system, and content structure.

The site uses Astro's content collections to manage projects and photography, with markdown files driving the layout of each project page. This means adding a new section is as simple as adding a new heading to the markdown file.

## Technical Approach

The architecture is built around the concept of "modes" — distinct visual experiences for different types of creative work. Each mode has its own colour scheme, typography, and layout conventions, all driven by CSS custom properties scoped to data attributes.

Content collections use Astro's glob loader to pick up markdown files, and the body of each file is rendered directly as the project detail page. This gives full flexibility over the structure of each project write-up without needing to touch any templates.

## Deployment

The site is deployed to Cloudflare Pages via a GitHub Actions workflow. Each push to main triggers a build and deploy, with Wrangler handling the deployment to the edge. -->
