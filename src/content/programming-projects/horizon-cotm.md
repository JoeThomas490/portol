## Description

I was moved over onto the Horizon project during the final push of its development. This was due to the team needing UI support to help bring it to its completed level of polish ready for release as a launch title for PSVR2.

## Challenges

Working in VR is a challenge for me as I suffer with motion sickness, so tracking down bug reproductions was never a fun task.

Another challenge was jumping onto a huge existing project and code base and having to understand how the systems I needed to fix interacted with each other. This required reaching out to engineers that had more experience with the code base to help talk me through existing systems.

Finally, working under the pressure of the final milestones to get ready for release was it’s own challenge, having to go back and forth with fixes to pass TRC and VRC checks, however working with a great team definitely helped throughout this process.
